# AI 欢迎页面组件

## 组件概述

`AIWelcomePage` 是一个功能丰富的 AI 聊天界面欢迎页面组件，支持亮色和暗色两种主题模式。该组件提供了美观的用户界面，包括科技蓝亚克力动态背景效果、滚动提示词、功能导航栏等元素。

## 主要功能

- **双主题支持**：完全支持亮色和暗色两种主题模式，自动适应系统设置
- **科技蓝亚克力背景**：亮色模式下的科技蓝亚克力动态背景效果，暗色模式下保留宇宙黑洞效果
- **滚动提示词**：自动滚动的提示词建议，鼠标悬停时暂停滚动，两侧有渐隐效果
- **透明导航**：完全透明无边框的导航栏，与背景融为一体
- **功能导航**：包含知识库、智能体、提示词模板、模型管理等功能入口
- **聊天历史**：侧边抽屉式聊天历史记录
- **AI 设置**：可配置的 AI 参数设置，如记忆功能、知识库关联、思考模式等
- **功能卡片**：提供提示词教程、大模型介绍、功能介绍等快捷入口
- **深色模式优化**：针对暗色主题进行全面优化，提供舒适的夜间使用体验
- **品牌标识**：使用自定义 SVG 格式的百灵 Logo，支持亮色和暗色主题
- **模型管理**：提供多种大语言模型选择，支持添加自定义模型
- **快捷模型切换**：在输入框底部可以快速切换使用的大语言模型
- **导航功能**：提供知识工程、智能体、提示词、模型管理和使用帮助等功能入口
- **提示词滚动展示**：自动滚动展示提示词建议，鼠标悬停时暂停
- **智能输入框**：支持快捷发送消息，集成知识库和思考模式功能

## 导航功能详情

### 知识工程

提供知识库管理功能，包括创建、查看和使用个人知识库。

### 智能体

展示和管理不同类型的AI智能体，如代码助手、数据分析师等。

### 提示词

提供提示词模板管理，帮助用户更高效地与AI交流。

### 模型管理

管理和切换不同的大语言模型，包括GPT-4、Claude 3、百灵-7B等。

### 使用帮助

提供全面的使用指南和教程，包括新手入门、高级技巧、提示词指南、知识库教程、模型对比和常见问题解答等内容，帮助用户更好地使用AI助手。

## 使用方法

```tsx
import AIWelcomePage from "./components/AIWelcomePage";

function App() {
  const handleSendMessage = (message: string) => {
    console.log("发送消息:", message);
    // 处理消息发送逻辑
  };

  return (
    <AIWelcomePage
      onSendMessage={handleSendMessage}
      onNewChat={() => console.log("新建聊天")}
      onSelectChat={(chatId) => console.log("选择聊天:", chatId)}
    />
  );
}
```

## 组件属性

| 属性           | 类型                      | 描述               |
| -------------- | ------------------------- | ------------------ |
| onSendMessage  | (message: string) => void | 发送消息的回调函数 |
| onToggleDrawer | () => void                | 切换抽屉的回调函数 |
| onOpenSettings | () => void                | 打开设置的回调函数 |
| onNewChat      | () => void                | 新建聊天的回调函数 |
| chatHistory    | Array                     | 聊天历史记录数组   |
| onSelectChat   | (chatId: string) => void  | 选择聊天的回调函数 |

## 品牌标识

组件使用自定义的百灵 Logo，具有以下特点：

- 使用 SVG 格式，确保在各种分辨率下都能清晰显示
- Logo 文件位置：`/public/images/logo-bailing.svg`
- 尺寸设置为 64x64 像素（w-16 h-16）
- 与"百灵2.0"文字标题搭配使用
- 在亮色和暗色主题下都能保持良好的可见性

## 主题适配

组件使用 Tailwind CSS 的暗色模式类实现主题切换，自动响应系统主题设置：

- **亮色模式**：

  - 纯白色背景配合科技蓝亚克力动态效果
  - 深色文本，提高可读性
  - 透明导航栏，与背景融为一体
  - 浅色卡片和输入框，带有轻微透明效果

- **暗色模式**：
  - 深灰色背景 (bg-gray-900)，取代原有的紫色渐变
  - 浅色文本，确保在暗背景上的可读性
  - 半透明深色卡片，带有模糊效果
  - 优化的色彩对比度，减少眼睛疲劳
  - 保留黑洞动画效果，增强视觉体验

## 深色模式优化

为提供更好的暗色主题体验，组件进行了以下优化：

1. **色彩系统调整**：

   - 使用深灰色 (gray-800/90) 替代纯黑色，减少对比度过高导致的视觉疲劳
   - 为功能按钮和卡片添加半透明效果，增强层次感
   - 优化文本颜色，使用 gray-200/300 代替纯白色，降低亮度

2. **UI元素优化**：

   - 卡片和按钮使用 backdrop-blur-sm 效果，增强在暗色背景上的可见性
   - 边框颜色调整为更柔和的深色调，减少突兀感
   - 提示词卡片使用紫色系 (purple-900/30)，而非原来的白色半透明

3. **渐变效果调整**：

   - 滚动提示词两侧渐变色调整为 rgba(17,24,39,0.95)，与深色背景更协调
   - 使用 @media (prefers-color-scheme: dark) 确保系统级暗色模式下也能正确显示

4. **交互反馈优化**：

   - 悬停状态使用更柔和的颜色变化 (dark:hover:bg-gray-700/50)
   - 激活状态使用深色主题专用的高亮色 (dark:bg-primary-900/50)
   - 按钮和卡片的阴影效果在暗色模式下更加微妙

5. **动画效果适配**：
   - 黑洞效果在暗色模式下保持，提供沉浸式体验
   - 粒子动画在暗色模式下调整为更适合夜间观看的亮度和对比度

## 科技蓝亚克力动态背景效果

科技蓝亚克力动态背景效果是使用Canvas实现的现代UI背景，具有以下特点：

1. **亮色模式特性**：

   - 纯白色背景上的蓝色网格线
   - 半透明蓝色几何粒子（圆形、方形、三角形）
   - 粒子之间的连接线，距离越近线条越明显
   - 粒子自由移动，超出边界后从另一侧重新进入

2. **暗色模式特性**：

   - 保留原有的宇宙黑洞效果
   - 黑洞中心使用径向渐变，从黑色核心向外扩散为紫色光晕
   - 粒子向黑洞中心移动，模拟被吸入效果
   - 优化的粒子亮度和对比度，适合夜间观看

3. **粒子系统**：

   - 多种几何形状的粒子
   - 随机大小、透明度和移动速度
   - 粒子之间的连接线形成网络效果
   - 粒子数量根据屏幕大小自动调整

4. **响应式设计**：
   - 画布大小会随窗口大小变化自动调整
   - 网格线和粒子分布会随画布大小变化
   - 粒子数量会根据屏幕大小自动调整，确保在不同设备上有良好的性能
   - 自动检测系统主题设置，切换相应的视觉效果

## 横向滚动提示词功能

在百灵2.0标题和输入框之间添加了2行横向滚动的提示词卡片：

1. **设计特点**：

   - 每行提示词自动横向滚动，相邻行滚动方向相反
   - 卡片高度不超过36px（实际设置为32px），确保紧凑布局
   - 提示词文本最多显示一行，超出部分使用省略号
   - 半透明背景和模糊效果，与整体UI风格一致
   - 两侧添加强烈渐隐效果（宽度为20%），使滚动更加自然流畅
   - 全宽自适应设计，提示词区域宽度为100%，随屏幕大小自动调整

2. **交互功能**：

   - 鼠标悬停在某一行时，该行自动停止滚动
   - 鼠标离开后，恢复自动滚动
   - 点击提示词卡片，自动将提示词填充到输入框中
   - 每行滚动速度略有不同，增加视觉层次感

3. **主题适配**：
   - 亮色模式：紫色系半透明卡片 (bg-purple-100/70)，深紫色文本
   - 暗色模式：深紫色半透明卡片 (dark:bg-purple-900/30)，浅紫色文本
   - 两侧渐变效果根据主题自动调整颜色
   - 使用 CSS 媒体查询确保系统级主题切换时也能正确显示

## 导航栏功能

完全透明的导航栏采用左右分离布局，左侧显示主要功能按钮（带图标和文字），右侧显示工具按钮（仅图标）：

1. **左侧功能按钮**：

   - **知识库**（图标+文字）：

     - 显示用户的知识库列表
     - 提供创建新知识库的入口
     - 同时显示数据库图标和"知识库"文字

   - **智能体**（图标+文字）：

     - 提供多种预设AI助手角色
     - 支持自定义智能体
     - 同时显示用户组图标和"智能体"文字

   - **提示词模板**（图标+文字）：

     - 提供常用提示词模板
     - 支持创建自定义模板
     - 同时显示模板图标和"提示词模板"文字

   - **模型管理**（图标+文字）：
     - 提供多种大语言模型选择
     - 显示各模型的特点和适用场景
     - 支持添加自定义模型
     - 同时显示芯片图标和"模型管理"文字

2. **右侧功能按钮**：

   - **会话列表**（仅图标）：

     - 打开会话历史抽屉，显示历史对话记录
     - 支持选择历史会话继续对话
     - 仅显示菜单图标，鼠标悬停时显示标题提示

   - **用户头像**（仅图标）：
     - 鼠标悬停或点击时显示下拉菜单
     - 菜单包含设置、反馈问题、数据三个选项
     - 仅显示用户头像图标，鼠标悬停时显示标题提示

## 设计特点

1. **沉浸式透明界面**：

   - 导航栏和内容面板采用完全透明背景，与科技蓝亚克力背景融为一体
   - 使用模糊效果（backdrop-blur）提高内容可读性，同时保持透明感
   - 无边框设计，增强视觉流畅性和现代感

2. **宽屏布局**：

   - 导航栏采用宽屏设计（max-w-5xl），提供更宽敞的操作空间
   - 左右分离布局，功能分组更加清晰
   - 主要功能按钮使用更宽的内边距，增强可点击区域

3. **增强可用性**：

   - 主要功能按钮同时显示图标和文字，提高可识别性
   - 工具按钮添加了标题提示（tooltip），确保用户理解其功能
   - 激活状态的按钮使用高亮颜色区分
   - 提示词卡片提供快速输入常用问题的便捷方式

4. **视觉层次**：

   - 导航栏使用模糊效果，在科技蓝亚克力背景上形成清晰的视觉层次
   - 激活状态的导航项使用高亮颜色区分
   - 横向滚动提示词增加页面动态感，同时不影响主要功能使用
   - 提示词两侧渐隐效果增强了视觉深度感

5. **用户个性化**：

   - 右侧用户头像提供个人设置和数据访问入口
   - 下拉菜单使用动画效果平滑显示
   - 提示词卡片提供个性化的问题建议

6. **响应式布局**：

   - 导航栏自适应不同屏幕尺寸
   - 在小屏幕设备上保持良好的可点击区域
   - 提示词卡片区域使用全宽设计，在各种设备上都能提供最佳体验

7. **深色模式优化**：
   - 所有UI元素都经过深色模式优化，确保在夜间使用时舒适
   - 使用半透明效果和模糊背景增强深色模式下的层次感
   - 色彩对比度经过精心调整，减少眼睛疲劳
   - 动画效果在深色模式下更加柔和

## 输入区功能

输入区位于页面中央，提供了丰富的交互功能：

1. **主输入框**：

   - 宽敞的输入区域，支持多行文本输入
   - 响应式设计，在各种设备上都有良好的使用体验
   - 按 Enter 键发送消息

2. **功能按钮区**：

   - **关联知识库**：切换是否使用知识库增强回答
   - **思考模式**：切换是否启用思考模式，提供更深入的分析
   - **模型选择**：快速切换使用的大语言模型
     - 显示当前选中的模型名称和图标
     - 点击展开下拉菜单，选择其他可用模型
     - 每个模型显示名称、图标和简短描述
     - 支持 GPT-4、Claude 3、百灵-7B、Gemini Pro、Llama 3 等多种模型
   - **附件按钮**：支持上传文件作为对话参考
   - **发送按钮**：发送当前输入的消息

3. **交互体验**：
   - 按钮状态变化提供明确的视觉反馈
   - 下拉菜单使用动画效果平滑显示
   - 点击页面其他区域自动关闭打开的下拉菜单
   - 适配亮色和暗色两种主题模式
